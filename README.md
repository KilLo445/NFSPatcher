# NFSUndergroundPatcher
Batch script that patches NFS Underground / Underground 2 to run on Windows 10

<!-- GETTING STARTED -->
## Getting Started

To patch Need for Speed Underground follow these simple steps.

### Installation

1. Download the [latest release](https://github.com/KilLo445/NFSUndergroundPatcher/releases/latest) corresponding to the game you want to patch and extract it.
  
2. Place the .bat file in your Need for Speed Underground install location.
   (Deafult is C:\Program Files (x86)\EA GAMES\Need for Speed Underground / C:\Program Files (x86)\EA GAMES\Need for Speed Underground 2)
   
3. Run it.
