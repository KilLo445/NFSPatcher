# NFSUndergroundPatcher
Batch script that patches NFS Underground to run on Windows 10

<!-- GETTING STARTED -->
## Getting Started

To patch Need for Speed Underground follow these simple steps.

### Installation

1. Download NFSUndergroundPatch.zip and extract it.
  
2. Place the .bat file in your Need for Speed Underground install location.
   (Deafult is C:\Program Files (x86)\EA GAMES\Need for Speed Underground)
   
3. Run it.
