# NFS Underground 2

Files for NFS Underground 2 patcher

If you experiance crashes, right click on the Underground 2 shortcut and delete everything in "Target".  
Copy and paste the text below into "Target", change "C:\Program Files (x86)\Need for Speed Underground 2\" to wherever you have the game installed.  
  
%WINDIR%\System32\cmd.exe /C CD /D "C:\Program Files (x86)\Need for Speed Underground 2\" && START /AFFINITY 1 SPEED2.EXE  
  
-----------------------------------------------------------------------------------------------------------------------------  
  
If you experiance frame drops, right click on SPEED2.EXE in your install location, and go to the "Compatibility" tab.  
Then enable "Compatibility Mode" and select "Windows XP (Service Pack 2)"  
  
Enjoy Underground 2!
